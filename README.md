LiveOn
======

2D Web Shooter created by Kareem Khattab, Patrick Ryan , Kevin Hou.
There is a JS, HTML and CSS File. 
To open, right click the html file, and hit "Open With Chrome". 
To Play click the Play Button on the Main Menu.
